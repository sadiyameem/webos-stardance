StardanceOS

# Description
- This is a browser-based operating system about Hack Clubs new program Stardance.

# Features
- Draggable apps and windows
- Dark mode
- Live clock
- Apps that open and close

# Tools
- HTML
- CSS
- JavaScript
- WebOS Guide provided
